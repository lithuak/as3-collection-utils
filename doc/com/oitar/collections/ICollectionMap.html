Property Detail
mapFunctionproperty
mapFunction:Function

A function that is applied to each item to perform an actual mapping. A mapFunction is expected to have the following signature: 

f(item:Object):Object
where the return value is any object for one-to-one mapping or ListCollectionViewResult instance for one-to-many mapping. The mapping function is mandatory and must be specified when calling the constructor. Later it can be changed by setting this property.
